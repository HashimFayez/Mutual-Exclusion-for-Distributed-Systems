1. Project2.java file can be compiled using following command:
	javac Project2.java

2. Execution needs nodeID as an argument while initiating a node!
	i.e. java Project2 0
	will initiate 1st node in the system.

3. config.txt file has information to initiate a node i.e. its nodeID, corresponding IP address and Port Number and number of Critical Section Requests that perticular node will generate.

4. quorumconfig.txt file has the quorum structure i.e. the shape in which nodes are arranged while picking a quorum set, how to pick column values as well as row values for any node.
